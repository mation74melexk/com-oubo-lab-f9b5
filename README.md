# com-oubo-lab-f9b5

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存储，方便直接访问或分发给需要的人。

## 项目简介

这是一个以 HTML 页面归档为主要目的的仓库。仓库内的每个页面都是独立的，彼此之间没有依赖关系。你可以通过 GitHub Pages 或其他静态托管服务来访问这些页面。

## 目录说明

```
├── index.html          # 仓库首页（可选，视实际内容而定）
├── page1/              # 示例页面目录
│   └── index.html
├── page2/              # 另一个示例页面目录
│   └── index.html
└── README.md           # 本文件
```

目录结构会根据实际归档的页面数量与类型进行调整。每个页面通常位于独立的文件夹中，便于管理和引用资源（如 CSS、JavaScript 文件）。
